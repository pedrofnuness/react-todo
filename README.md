## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/pedrofnuness/react-todo.git
   ```
3. Install packages
   ```sh
   yarn
   ```
4. Run in development mode
   ```sh
   yarn dev
   ```
